[[X Road - Central Services]]

Os serviços centrais consistem no Servidor Central e no Proxy de Configuração. O Servidor Central contém o registro dos membros do X-Road e seus Servidores de Segurança. Além disso, o Servidor Central contém a política de segurança da instância do X-Road, que inclui uma lista de autoridades de certificação confiáveis, uma lista de autoridades de registro de data e hora confiáveis ​​e parâmetros de configuração. Tanto o registro de membros quanto a política de segurança são disponibilizados aos Servidores de Segurança por meio dos protocolos HTTP e HTTPS. Esse conjunto distribuído de dados forma a configuração global que os Servidores de Segurança utilizam para intermediar as mensagens enviadas pelo X-Road. O Operador do X-Road é responsável pela operação do Servidor Central. 

O Proxy de Configuração é um componente opcional que pode ser usado como um intermediário para publicar a configuração global nos Servidores de Segurança para download. O Proxy de Configuração primeiro baixa a configuração global do Servidor Central e, em seguida, a distribui de forma segura. O Proxy de Configuração pode ser usado para aumentar a disponibilidade do sistema, criando uma fonte de configuração adicional e reduzindo a carga no Servidor Central. O Operador do X-Road é responsável por operar o Proxy de Configuração.
