# Newbify X-Road

Repositório colaborativo para compartilhar códigos, referências técnicas e anotações sobre o ecossistema **X-Road**, com foco em integração e desenvolvimento de serviços.

## Conteúdo

- Fundamentos: arquitetura, componentes, identificadores, protocolos e segurança.
- Desenvolvimento de serviços: Security Server, REST, SOAP e troubleshooting.
- Termos, traduções, relatórios, imagens e materiais de estudo.
- Exemplos de código e configurações úteis para o time.

As anotações são mantidas como um vault do Obsidian. O mapa principal de estudo está em `.obsidian/4 - Notas Principais/X-Road/X-Road - MOC/MOC - X-Road.md`.

## Contribuição

- Escreva notas objetivas em Markdown e conecte assuntos relacionados quando possível.
- Inclua contexto e instruções de uso em exemplos de código.
- Não envie senhas, chaves privadas, certificados ou dados de ambientes reais.
- Descreva resumidamente no pull request o que foi incluído ou alterado.
