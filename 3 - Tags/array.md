Um array (vetor ou arranjo) ==é uma estrutura de dados fundamental que armazena uma coleção sequencial de elementos do mesmo tipo sob um único nome de variável==. Os itens são organizados em posições contíguas de memória e acessados por índices numéricos, geralmente começando em zero. 

**Características Principais:**

- **Indexação:** Cada elemento possui uma posição numérica (índice) que permite acesso rápido.
- **Homogeneidade:** Geralmente, todos os elementos devem ser do mesmo tipo de dado (inteiros, strings, etc.).
- **Tamanho Fixo:** Em muitas linguagens (como C++), o tamanho do array é definido no momento da criação e não pode ser alterado.
- **Armazenamento Contíguo:** Elementos são guardados em células de memória adjacentes, facilitando a manipulação e acesso rápido. 

**Exemplo (Conceitual):**  
Um array chamado `notas` com 3 elementos:  
`[8.5, 9.0, 7.5]`
