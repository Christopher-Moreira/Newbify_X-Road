[[efficient retrieval]]
São metodos, algoritmos ou infraestruturas projetadas para localizar infos relevantes em um conjunto de dados massivos nao estruturados
## O que é Efficient Retrieval 
É recuperar a informação certa, o mais rápido possível, com o menor custo computacional, mesmo em grandes volumes de dados.

	Buscar só os textos semanticamente relevantes
	Enviar apenas os top 3–5 trechos para o modelo
